Fast, intelligent, and fully customizable search for your site. Comes with detailed analytics and controls in the Tinysou Dashboard.

== Description ==

The Tinysou Search plugin replaces the standard WordPress search with a better, more relevant search engine. It also gives you detailed insight into what your users are searching for, so you know which keywords to target when customizing your search engine results. The Swiftype search plugin is WordPress VIP-approved and already used on huge sites.  The search plugin is complemented by our dashboard, [full-featured developer API](http://doc.tinysou.com/v1/overview.html), Manage search results with drag and drop and see the changes reflected instantly.
